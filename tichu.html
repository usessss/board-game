👇아래 링크를 통해 온라인에서 티추를 플레이할 수 있습니다 ❣

🐉 티추 게임하러 가기 🐦‍🔥

🐉 티추 완전 정리 :: 핵심 규칙 + 전략 + 꿀팁

🎯 게임 목표 ▼
  • 2:2 팀전 카드 게임
  • 점수를 모아 1000점 먼저 도달하는 팀이 승리
  • 한 판에는 항상 100점이 존재
🃏 카드 구성 ▼
52장 일반 카드 + 4장 특수카드
  • 🪶 참새(1) : 라운드 시작 / 원하는 숫자 선언 가능
  • 🐶 강아지 : 턴을 팀원에게 넘김
    ( 4명 모두 있다면 팀원에게 선을 줌, 원카드 J 라고 생각하면 됨 )
  • 🐦‍🔥 피닉스(봉) : 만능 카드, 대신 -25점
  • 🐉 드래곤(용) : 싱글 중 최강, 점수는 상대에게 +25
🔥 낼 수 있는 패 (족보) ▼
  • 싱글 / 페어 / 트리플
  • 스트레이트 ( 최소 5장 )
  • 연페어 ( 22 33, 33,44,55 가능 )
  • 폭 : 게임에서 가장 강한 패( 같은 숫자 4장 < 같은 색깔 5장 이상 )
💥 폭 완전 이해하기 ▼
폭은 어떤 패보다 무조건 우선❗

폭 종류 2가지
  • 같은 숫자 폭 ( 예 : 7·7·7·7 )
  • 색깔 스트레이트 폭 ( 예: ♣ 2-3-4-5-6 )

강함 순위
  • 같은 숫자 4장 < 5장 < ...
  • 색깔 폭이 숫자 폭보다 무조건 더 강함
✅ 같은 장수인데 숫자가 높으면 더 쎔
✅ 색깔 스트레이트 폭도 마찬가지로 높은 숫자거나 장수 많으면 더 쎔
📌 점수 규칙 ▼
  • 5 = +5점
  • 10 = +10점
  • K = +10점
  • 🐉 드래곤 = +25점
    👉 용으로 먹은 건 상대 팀( 꼴찌할 것 같은 사람 )에게 줘야 함
  • 🐦‍🔥 피닉스 = -25점 ( 봉은 0.5, A 밟을 수 있음 )
📣 스티 / 라지 티추 선언 ▼
  • 스티 (성공 시 +100/실패 시 -100) : 카드 14장 보고 선언
  • 라티 (성공 시 +200/실패 시 -200) : 카드 8장 보고 선언
🔄 카드 3장 교환 전략▼
  • 팀원이 라티 선언했다면 → 무조건 탑 패 보내기, 강아지는 내가 갔고있기
  • 상대가 라티 선언했다면 → 좋은 카드는 절대 주지 않기
    ( 낮은 카드 위주로 주기, 폭 될 수도 있으니 낮은 숫자 페어 찢어주면 좋음 )
  • 강아지
    팀원이 라티면 갔고있고 아니라면 내 오른쪽 상대팀에게 주는 게 좋음
    ( 그래야 내가 그 다음으로 턴을 빨리 가져올 수 있음 )
  • 드래곤
    내가 탑패 2장 이상 보유하고 있거나 스티 각 보고 있다면 갖고있기
    드래곤으로 선 잡으면 그 패들은 상대팀 중 꼴찌할 것 같은 사람에게 줘야함
    ( 꼴찌 패는 1등한테 전부 가기 때문 ) ✅ 드래곤은 싱글 중 탑패라 A를 밟은 봉도 밟을 수 있음
  • 피닉스
    내가 탑패 2장 이상 보유하고 있거나 스티 각 보고 있다면 갖고있기
    스트레이트가 있다면 우리팀에게 주기( 이빨 빠진 부분이 있을 수도 있기 때문 )
🚦 턴/맞받아치기 규칙 ▼
  • 같은 패 종류로만 맞받아칠 수 있음
  • 싱글 → 싱글, 연페어 → 연페어, 트리플 → 트리플, 스트레이트 → 스트레이트
🤝 팀플 핵심 전략 ▼
  • 참새'1' 가진 사람은 내가 필요한 숫자 "소원" 빌 수 있음
    ⚠ 다음 턴인 상대가 가진 패가 확실하지 않아서
    그 콜이 자칫 우리 팀에게 갈 수 있으니 신중 ⚠
    참새를 가진 사람이라고 해서 무조건 소원 빌지 않아도 됨
    참새 먼저 안 내고 선을 잡을 수 있다면 다른 카드를 내도 무관
    참새 한 장만 내거나 참새를 포함한 1, 2, 3, 4, 5 스트레이트도 가능
    🔪 팀킬 하지 않기 ( 무섭다면 보통은 준 거 소원 빔 ) 🔪

  • 우리 팀이 라티면 강아지는 내가 갖고 있는 게 좋음
    팀원이 선을 못 잡는 상황이라면 내가 선을 잡고
    강아지로 팀 선 주고 서포트 해줘야 함
    💫 라티 성공할 수 있게 잘 서포트 해주기 💫

  • 🙅‍♀️ 우리 팀이 낸 알파벳은 웬만하면 밟지 않기 🙅‍♀️
    숫자면 털 패 후다닥 내기
💡 실전 꿀팁 ▼
  • 폭은 너무 아끼면 손해
  • K는 쎈 패라 A 4장이 다 빠졌을 때 싱글로 내서 선 먹기 중요
  • 용은 절대 막 쓰지 말기
    ( 한방에 나가는 게 아니라면 A 위에 쓰기 )
  • 피닉스는 페어 강화나 스트레이트 보완에 좋음
  • 팀이 1등·2등 먹으면 → 200점
    패교환 했을 때 우리 팀이 낮은 패 서로 교환했다 하면 원투 각 보기
  • 봉은 -25점 이기에 용 봉 두 장을 들고 있다면
    용 먼저 내고 선 먹고 봉으로 나가는 게 좋음
    그냥 맨 바닥에 두면 숫자 2보다 약한 0.5 취급 당함
    -25점 먹기 싫어서 이렇게 하는 것
    🐦‍🔥 봉은 단독으로 내든, 다른 점수 없는 카드와 묶어서 내든
    가능하면 마지막에 내고 빠져나가는 게 가장 좋다 🐦‍🔥
