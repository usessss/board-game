🎯 게임 목표

2 vs 2 팀전으로 진행되며, 더 높은 점수를 먼저 달성한 팀이 승리합니다.
패를 빨리 모두 내거나, 점수 카드를 많이 먹어 점수를 모읍니다.

🃏 카드 구성

  • 일반 카드 52장
  • 특수 카드 4장 — 드래곤, 피닉스, 도그, 마작(1)

➡️ 기본 진행 방식

  1. 카드 8장씩 받고 1차 선택(티추 선언 가능)
  2. 나머지 6장을 추가로 받아 총 14장 보유
  3. 각자 원하는 사람에게 카드 3장씩 전달
  4. 선 플레이어부터 카드 패턴을 맞춰 내기 시작
  5. 모든 카드를 먼저 소진하면 라운드 종료

📌 낼 수 있는 패 종류

  • 단독 1장
  • 같은 숫자 세트 (페어, 트리플, 포카드)
  • 연속 숫자 스트레이트 (예: 4-5-6-7-8)
  • 페어 스트레이트 (예: 3-3 / 4-4 / 5-5)

→ 같은 종류의 패로만 맞받아칠 수 있어요!

🔥 특수 카드 설명

  • 🀄 마작(1) — 라운드 첫 시작 가능 / 원하는 숫자 부를 수 있음
  • 🐶 도그 — 다음 턴을 우리 팀 동료에게 넘김
  • 🕊 피닉스 — 어떤 패에도 추가 가능 / -25점 페널티
  • 🐉 드래곤 — 가장 강한 단독 카드 / 먹은 점수는 상대에게 줌

🏁 점수 계산

  • 5 = 5점
  • 10 = 10점
  • K = 10점
  • 드래곤 = +25점 (상대에게 줌)
  • 피닉스 = -25점

→ 총 100점이 존재하며, 1000점 먼저 달성하면 승리!
